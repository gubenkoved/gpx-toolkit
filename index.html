    Demo mode — using a simulated phone so you can explore without hardware.
    Click Connect phone to drive a real Android phone over WebUSB (Chromium, over HTTPS or
    localhost, with no local adb server running). Android only — iOS/iPhone is not supported.
    

      How do I enable ADB on my Android phone?
      
    
        
  1. Open Settings → About phone and tap Build number seven times until you see
          “You are now a developer”.
    2. 
        
  2. Go to Settings → System → Developer options and turn on USB debugging.
          (On some phones it's Settings → Developer options.)
    3. 
        
  3. Connect the phone to this computer with a USB cable.
    4. 
        
  4. Click Connect phone above and pick your device in the browser's chooser.
    5. 
        
  5. On the phone, accept the “Allow USB debugging?” prompt (tick Always allow
          to skip it next time).
    6. 
      

      
Connection fails with “device busy”? A local adb server is holding the phone
        — run adb kill-server in a terminal, then click Connect phone again.
